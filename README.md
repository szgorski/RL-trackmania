# Reinforcement Learning Pipeline Customization (for TrackMania)

This project explores **customization** of a reinforcement learning (RL) training pipeline based on the TMRL framework, focusing on adapting model architectures and training logic. The goal of a project is to understand and extend an existing RL system.

## Contribution

Contribution of authors:

- modification of Soft Actor-Critic implementation,
- adjustment of loss computation and update flow,
- tuning hyperparameters and training behavior,
- testing MLP-based against RNN-based actors.

**The codebase is not built from scratch.**
